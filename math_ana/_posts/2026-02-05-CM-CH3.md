# 引言

前面讨论的是“抽象”的集合，即由任何对象构成的集合（可以是任何对象）。本章我们专门研究实数集合、复数集合以及**更一般地研究高维空间中的集合**（量子力学会涉及到的一些概念）。本章中我们会经常提到实线上的点集、平面上的点集，或某个**高维空间**中的点集。

### 1. 欧氏空间$\textbf{R}^n$

二维空间中的一个点是一个实数序偶$(x_1, x_2)$。类似地，三维空间中的一个点是一个有序三元实数组$(x_1, x_2, x_3)$。现在容易想到一个有序的$n$元实数组$(x_1, x_2, \cdots, x_n)$，并将其称为$n$维空间中的一个点。

**定义** 设$n>0$是一个整数。一个由$n$个实数组成的有序集合$(x_1, x_2, \cdots, x_n)$称为一个$n$维点，或称为一个有$n$个分量的向量。点或向量通常用单个的黑体字母表示，例如，

$$\textbf{x}=(x_1,x_2,\cdots , x_n),\quad \text{或}\quad \textbf{y}=(y_1,y_2,\cdots , y_n)$$

数$x_k$称为点$\textbf{x}$的第$k$个坐标，或称为向量$\textbf{x}$的第$k$个分量。由**全体$n$维点组成的集合称为$n$维欧氏空间**，或简称为$n$维空间，并记为$\mathbf{R}^n$。

那么，讨论维数大于3的空间有啥好处呢？$n$维空间的语言可以使许多复杂的情况变得非常容易被理解。经过高中物理，我们对三维向量足够了解，于是了解有三个自由度的系统写成一个向量运动方程而不是三个标量方程的优越性，当一个系统有$n$个自由度时也有类似的优越性。并且，还可以让我们能够处理与空间维数无关的性质。

关于$n$维点的代数运算定义如下：

**定义** 设$x=(x_1, x_2, \cdots, x_n)$和$y=(y_1, y_2, \cdots, y_n)$属于$\textbf{R}^n$。我们定义：

a) 相等：

$$\textbf{x}=\textbf{y}\quad\text{当且仅当}\quad x_1=y_1, \cdots, x_n=y_n.$$

b) 和：

$$\textbf{x}+\textbf{y}=(x_1+y_1, \cdots, x_n+y_n).$$

c) 实数（标量）乘法：

$$a\textbf{x}=(ax_1, \dots, ax_n)\quad(a\text{是实数}).$$

d) 差：

$$\textbf{x}-\textbf{y}=\textbf{x}+(-1)\textbf{y}.$$

e) 零向量或原点：

$$\mathbf{0}=(0, \dots, 0).$$

f) 内积或点积：

$$\textbf{x}\cdot \textbf{y}=\sum_{k=1}^n x_ky_k.$$

g) 范数或长度：

$$\mid\mid x\mid\mid =(x\cdot x)^{1/2}=\left(\sum_{k=1}^n x_k^2\right)^{1/2}.$$

范数$\mid\mid x-y\mid\mid $称为$x$和$y$之间的距离。

**定义** $\textbf{R}^n$中的单位坐标向量$\textbf{u}_k$是第$k$个分量为1而其余分量均为零的向量。因此，

$$\textbf{u}_1=(1, 0, \cdots, 0),\quad \textbf{u}_2=(0, 1, 0, \cdots, 0),\cdots,\quad \textbf{u}_n=(0, 0, \cdots, 0, 1).$$

如果$\textbf{x}=(x_1, x_2, \cdots, x_n)$，则有$\textbf{x}=x_1\textbf{u}_1+\cdots+x_n\textbf{u}_n$和$\textbf{x}_1=\textbf{x}\cdot \textbf{u}_1,\, \textbf{x}_2=\textbf{x}\cdot \textbf{u}_2,\,\cdots,\,\textbf{x}_n=\textbf{x}\cdot \textbf{u}_n$。向量$\textbf{u}_1,\,\textbf{u}_2,\,\cdots,\,\textbf{u}_n$也称为基向量。

### 2. $\textbf{R}^n$中的开球与开集

设$a$是$\mathbf{R}^n$中的一个给定的点，$r$是一个给定的正数。由$\mathbf{R}^n$中所有满足条件

$$\mid\mid \textbf{x}-\textbf{a}\mid \mid< r, $$

的点$x$构成的集合称为**半径为$r$、中心为$a$的开的$n$-球**（不一定是个三维空间的球），我们用$B(a)$或$B(a; r)$来表示这个集合。

球$B(a; r)$由全部到$a$的距离小于$r$的点组成。在$\textbf{R}^1$中这只是一个以$a$为中心的开区间，在$\textbf{R}^2$中它是一个圆盘，而在$\textbf{R}^3$中它是中心在$a$点、半径为$r$的球体。

**定义** 设$S$是$\mathbf{R}^n$的一个子集，并且假定$a\in S$。如果有一个中心在$a$点处的开的$n$-球，它的每一个点都属于$S$，则$a$称为$S$的一个内点。

换句话说，$S$的每一个内点$a$可以被一个$n$-球$B(a)\subseteq S$环绕。由$S$的全部内点组成的集合称为$S$的内部，并记之为$\text{int}S$（反之就是那些处于$S$表面的点，那些点不能作为一个球的球心）。包含一个以$a$为中心的球的任意集合有时称为$a$的一个邻域。

**定义（开集的定义）** 如果$\mathbf{R}^n$中集合$S$全部点都是内点，则称$S$为开集。

**注：**$S$是开集当且仅当$S={\rm int}S$。

**注意：**$\textbf{R}^1$内的开区间当作为平面的一个子集考虑时则不再是开集（因为一条线在二维空间内，线的宽度也要考虑）。事实上，$\textbf{R}^1$中没有子集（空集除外）能在$\textbf{R}^2$内是开集，因为这样的集合不可能包含2-球。

**定理** 任何开集族的并集都是开集。

在$\mathbf{R}^n$中空集是开集，就像整个空间$\mathbf{R}^n$是开集一样。每个$n$-球都是$\mathbf{R}^n$中的开集，$n$个一维开区间$(a_1, b_1), \cdots, (a_n, b_n)$的笛卡儿积

$$(a_1, b_1) \times \cdots \times (a_n, b_n)$$

是$\mathbf{R}^n$中的一个开集，称为$n$维开区间，记为$(\textbf{a}, \textbf{b})$，其中$\textbf{a}=(a_1, \cdots, a_n),\, \textbf{b}=(b_1, \cdots, b_n)$。  

**定理** 任何开集族的并集都是开集。

**证明** 设$F$是一开集族，$S$表示它们的并，$S=\bigcup_{A\in F}A$。假定$x\in S$，则$x$必定至少属于$F$内各集合中的一个，比方说$x\in A$。因为$A$是开的，所以存在开的$n$-球$B(x)\subseteq A$。但是$A\subseteq S$，所以$B(x)\subseteq S$，从而$x$是$S$的内点。由于$S$的每一个点都是内点，所以$S$是开集。

**定理** 开集有限族的交是开集。（**有限族**：有限个集合组成的族。）

### 3. $\textbf{R}^1$中开集的结构

**定义 （构成区间）** 设$S$是$\mathbf{R}^1$的一个开子集。如果一个开区间$I\subseteq S$，而且没有开区间$J\neq I$使得$I\subseteq J\subseteq S$，则$I$(可以是有限的或无限的)称为$S$的一个构成区间。

想象实数轴是一根长绳子：

- 开集是绳子上**被涂色的一些段落**

- **构成区间**就是这些涂色段落中**连续的、不中断的一段段**

- 不同涂色段落之间有空白的绳子隔开

**定理** 非空开集$S$的每一个点都属于且只属于$S$的一个构成区间。

**定理** $\textbf{R}^1$内每一个非空开集$S$都是由$S$的互不相交的构成区间构成的可数族的并集。

### 4. 闭集

**定义 (闭集的定义)** $\mathbf{R}^n$内的集合$S$称为闭集，如果它的余集$\mathbf{R}^n - S$是开集。

例 $R^1$中的闭区间$[a, b]$是闭集。$n$个一维闭区间的笛卡儿积
$$[a_1, b_1] \times \cdots \times [a_n, b_n]$$
是$\mathbf{R}^n$中的闭集，称为$n$维闭区间$[a, b]$。

**定理** 闭集的有限族的并集是闭集，闭集的任意族的交集是闭集。

**定理** 如果$A$是开集而$B$是闭集，则$A-B$是开集而$B-A$是闭集。

**证明** 我们只需注意$A-B = A \cap (\textbf{R}^n - B)$，它是两个开集的交集；$B-A = B \cap (\textbf{R}^n - A)$，它是两个闭集的交集。

### 5. 附贴点，聚点

闭集也可以用附贴点和聚点来描述。

**定义 (附贴点的定义)** 设$S$是$\mathbf{R}^n$的一个子集，$x$是$\mathbf{R}^n$内的一个点，$x$未必在$S$内。如果每个$n$-球$B(x)$都至少包含$S$的一个点，那么称$x$为$S$的一个附贴点。

**例**

1. 如果$x\in S$，则$x$附贴于$S$，因为每个$n$-球$B(x)$都包含$x$。

2. 如果$S$是$\mathbf{R}$的一个有上界的子集，则$\sup S$附贴于$S$。

有些点附贴于$S$是由于每个球$B(x)$都含有$S$的异于$x$的点，这些点称为聚点。

**定义 (聚点的定义)** 设$S\subseteq\mathbf{R}^n$, $x\in\mathbf{R}^n$，如果每个$n$-球$B(x)$至少包含$S$的一个与$x$不同的点， 则$x$称为$S$的聚点。

换句话说, $x$是$S$的一个聚点, 当且仅当$x$附贴于$S-\{x\}$。如果$x\in S$但$x$不是$S$的聚点, 则称$x$为$S$的孤立点。

**例**

1. 由形如$1/n, \, n=1, \, 2, \, 3, \, \cdots$的数组成的集合以0为聚点。
2. 有理数集合以每一个实数为聚点。
3. 闭区间$[a, \, b]$的每一个点都是由开区间$(a, \, b)$内的数组成的集合的聚点。

**定理** 如果$x$是$S$的聚点，则每一个$n$-球$B(x)$都包含$S$的无穷多个点。

### 6. 闭集与附贴点

闭集被定义为开集的余集，下一个定理用另一种方式描述了闭集。

**定理** $\textbf{R}^n$中的集合$S$是闭集，当且仅当它包含它的全部附贴点。

**定义 (闭包的定义)** 由集合$S$的全部附贴点组成的集合称为$S$的闭包，记为$\bar{S}$。

对于任何集合都有$S \subseteq \bar{S}$，因为$S$的每个点都附贴于$S$。上面的定理表明，相反的结论$\bar{S} \subseteq S$当且仅当$S$是闭集时成立。因此我们有：

**定理** 集合$S$是闭集当且仅当$S = \bar{S}$。

**定义 (导集的定义)** 由集合$S$的全部聚点组成的集合称为$S$的导集，记为$S'$。

显然，$\bar{S} = S \cup S'$对任一集合$S$成立。于是定理3.20意味着$S$是闭集当且仅当$S' \subseteq S$。换句话说，我们有以下定理：

**定理** $\textbf{R}^n$内的集合$S$是闭集，当且仅当它包含它的全部聚点。
